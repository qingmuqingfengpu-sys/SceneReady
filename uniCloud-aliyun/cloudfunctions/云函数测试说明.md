# 云函数测试说明

## geo-service 云函数

### 测试方法
右键 `geo-service` -> "上传并运行"

### 测试参数 (geo-service.param.json)

**逆地理编码**:
```json
{
  "action": "regeo",
  "longitude": 116.397470,
  "latitude": 39.908823
}
```

**地理编码**:
```json
{
  "action": "geo",
  "address": "北京市朝阳区",
  "city": "北京"
}
```

**POI搜索**:
```json
{
  "action": "search",
  "keywords": "咖啡厅",
  "city": "北京"
}
```

## order-co 云对象

### 测试方法
右键 `order-co` -> "上传并运行"

### 测试参数 (order-co.param.json)

**测试 Token**:
```json
{
  "$clientInfo": {
    "methodName": "testToken"
  }
}
```

**获取订单列表** (需登录):
```json
{
  "$clientInfo": {
    "methodName": "getMyOrders",
    "uniIdToken": "你的token"
  },
  "page": 1,
  "pageSize": 10
}
```

### 获取 Token

在小程序控制台执行:
```javascript
console.log(uni.getStorageSync('uni_id_token'))
```
